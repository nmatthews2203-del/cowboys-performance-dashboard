# Dallas Cowboys Performance Analytics Dashboard (1999–2025)

## Overview

This project explores historical performance of the **Dallas Cowboys** from **1999 to 2025** using Excel for data preparation and Tableau for interactive visualization.

The dashboard enables users to analyze:

* Win/loss records against opponents
* Division performance trends
* Season-level point differential
* Historical performance patterns across multiple filters

---

## Tools Used

* **Excel** — data cleaning, feature engineering, and metric creation
* **Tableau** — interactive dashboard design and visualization

---

## Key Features

* Interactive filters for **season, stadium, quarterback, game type, and home/away**
* Opponent-level **win/loss percentage analysis**
* Division performance **treemap visualization**
* Combined **wins/losses + point differential trend** over time
* Summary KPIs including total **wins, losses, and ties**

## Data Preparation Process

1. Filtered historical NFL game data to isolate **Dallas Cowboys matchups (1999–2025)**
2. Engineered analytical columns in Excel:

   * Opponent identification
   * Dallas score vs opponent score
   * Point differential
   * Win/loss/tie classification
3. Built calculated fields and interactive visuals in Tableau to support **exploratory sports analytics**.

---

## Project Purpose

This project demonstrates an **end-to-end analytics workflow**:

**Raw data → cleaning in Excel → visualization in Tableau → interactive insights**

It serves as a portfolio piece for **data analytics and sports analytics opportunities**.
